## Virtual Environment

### Activate
```
python -m venv venv
source ~/venv/bin/activate
```

### Deactivate
```
deactivate
```


## Installation 

Installation des dépendances
```
pip install -r requirements.txt
```

Installation de bootstrap
```
cd GestionProduit/app/static/
npm install
```

## Migration
Depuis la racine du projet, effectuez :
```
cd GestionProduit
python3 manage.py migrate
```
